# Book Management REST API

A beginner-friendly Book Management REST API built using Spring Boot.

## Tech Stack
- Java
- Spring Boot
- Spring Data JPA
- MySQL
- Maven

## Features
- Create Book
- Get All Books
- Get Book By ID
- Update Book
- Delete Book
- Validation
- Global Exception Handling

## API Endpoints
POST /books  
GET /books  
GET /books/{id}  
PUT /books/{id}  
DELETE /books/{id}

## How to Run
1. Clone the repository
2. Configure database in application.properties
3. Run the application
